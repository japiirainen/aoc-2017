# aoc-2017 in `Agda`

The point of this is to learn to program in `Agda` while solving fun coding puzzles.

## How to use

### Build

```sh
nix build #aoc
```

### Run build artifacts

Run all solutions:

```sh
./result/bin/aoc
```


Run a specific day and part solutions:

```sh
./result/bin/aoc --day 2

./result/bin/aoc --day 2 --part 2
```
